# liri-node-app
 LIRI is a Language Interpretation and Recognition Interface. It's a command line node app that takes in parameters and gives you back data.
 
 Technologies Utilized
*NodeJS
*JavaScript
*Spotify API
*OMDB API
*NPM dotenv
*NPM request
*NPM moment
